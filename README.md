# Paisabazaar Credit Score Analysis 

### **Project Overview**
This project analyzes customer financial data to identify the key drivers behind credit scores. Using Python, I performed data cleaning, exploratory data analysis (EDA), and provided strategic business solutions to improve creditworthiness.

### **Key Highlights**
* **Data Wrangling:** Handled missing values and treated outliers using the IQR method.
* **EDA:** Created 12 high-impact visualizations covering Univariate, Bivariate, and Multivariate analysis.
* **Business Insights:** Identified that payment behavior and debt-to-income ratios are more critical than raw salary.

### **Tech Stack**
* **Language:** Python
* **Libraries:** Pandas, NumPy, Matplotlib, Seaborn, Scipy
* **Platform:** Google Colab / GitHub
